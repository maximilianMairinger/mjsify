# Mjsify

Make an ESM file explicit (.mjs), on all filenames and import statements.

> Please note that Mjsify is currently under development and not yet suited for production

## Installation

```shell
 $ npm i mjsify
```

## Usage



```ts
import mjsify from "mjsify"

mjsify()
```
